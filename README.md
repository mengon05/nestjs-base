## Description

Test project based on the udemy course [NestJS from zero to hero](https://www.udemy.com/course/nestjs-zero-to-hero)

## Installation

```bash
$ npm install
```

Initialize postgres and pgAdmin4

```bash
$ sh init-pg-docker.sh
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
